1.2.5 Project - Super Fun Cup Game

This game creates three cups, either blue or red, depending on the user's color choice and randomly selects one of the three cups to be special. The cups then shuffle and the user has two tries to guess which cup is special. If they guess wrong, one of the two hearts on the screen dissapears. Once they lose both hearts, they lose the game and it ends. If they guess the correct cup on the first or second try, then they win.

3.1.6 Project - Rover Phone Home

A data recording rover got lost and scientists want to see if they can figure out where it is so they can retrieve and fix it. The rover recorded and sent data back about where it was, including temperature sound, wind speed, and light. The data was sent as an analogous sata set so we separated the data based on values and decimal places and then tried to firuge out which data set corresponded to which element. We found that the rover was somewhere with constant temperature, a small amount of wind, and constant sounds from animals, which matched the description of the marsh land.


3.2.4 Project - Making Meaning from Data

We were given a huge data set and were trying to find patterns with the honey production of each state. We cleaned the data so we only had the data we absolutely needed. Then we used Python to further sort and combine values in the list so we could plot them on matplotlib graphs in the program. We did this a few different ways and ended up with a lot of different graphs expressing the total, mean, and yearly production of honey.


4.1.4 Project- Understanding Complex Systems .

We looked at a simulation on NetLogo that showed two patients with bacterial infections. We could control how often the bacteria reproduced as well as how much and how often antibiotics were given to kill the bacteria. Randomness was used to place the bacteria, determine how many holes the bacteria had in the cell membrane, and how the antibiotics floated around in the bloodstream. However the simulation did not include whether or not the patients took care of themselves, how strong their immune systems were, or how bacteria can adapt to be resistant to antibiotics.
